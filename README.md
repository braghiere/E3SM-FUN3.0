 # E3SM-FUN 

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20452251.svg)](https://doi.org/10.5281/zenodo.20452251)

This contains data files used in [Braghiere et al. (2022)](https://agupubs.onlinelibrary.wiley.com/action/showCitFormats?doi=10.1029%2F2022MS003204).

##  Table of contents

* E3SMv1-FUN3.0.patch - A unix patch file for E3SM. The E3SM code is available from the [E3SM repository](https://e3sm.org/model/running-e3sm/e3sm-quick-start).
* files/ELM_FUNP.clm2.h0.1994-2005.timemean.nc.gz - Time-mean (1994–2005) ELM-FUN3.0 model output (gzip compressed). Contains global carbon, nitrogen, and phosphorus cycle variables averaged over the 1994–2005 period. Decompress with: `gunzip ELM_FUNP.clm2.h0.1994-2005.timemean.nc.gz`
* **Full monthly output (1994–2005, 2.2 GB compressed):** The complete time-series `ELM_FUNP.clm2.h0.1994-2005.tar.gz` is archived on Zenodo: https://doi.org/10.5281/zenodo.20452251. Decompress with: `tar -xzf ELM_FUNP.clm2.h0.1994-2005.tar.gz`
